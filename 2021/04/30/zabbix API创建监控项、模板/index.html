一、zabbix API简介

zabbix开放了一套完善的API，可用于二次开发、API自动化操作等，zabbix API是基于json-rpc2.0协议的web API，即客户端对API的请求和响应使用json格式编码。Zabbix API由许多名义上分组的独立API方法组成。大多数API至少包含四种方法： get， create， update 和 delete。

zabbix5.0手册现已支持中文，官方手册地址:zabbix 官方中文手册
